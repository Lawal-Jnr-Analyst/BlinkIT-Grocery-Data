This project features a comprehensive sales dashboard for Blinkit, India's last-minute delivery app. By analyzing transactional data across multiple outlet types, locations, and product categories, the dashboard identifies key drivers of revenue, including fat content distribution, item type performance, outlet size contribution, and regional tier breakdowns.

## ğŸ” Problem, Action & Result
- The Problem: Blinkit had inconsistent sales performance across outlet types and locations, but lacked visibility into which segments were underperforming and why margins varied significantly across tiers.
- The Action: I built DAX measures to calculate average sales, total revenue, and item visibility by outlet type, and used Power Query to clean and segment the data by fat content, item category, and outlet establishment year. These were visualized in an interactive Power BI dashboard with dynamic filter panels.
- The Result: Identified that Supermarket Type 1 accounts for $787.55K (65%+) of total sales despite similar average ratings across all outlet types, and revealed that Tier 3 locations outperform Tier 1 by over $135K in total sales, highlighting a significant underinvestment in lower-tier market expansion.
